3: Jak je možné propojit JavaScript s webovou stránkou?

Doplňte další možnosti ve formě kódu.

Opatřete kód poznámkami!

// 1. Vložit kód do tagu script
<script>console.log('Hello');</script>
// 2. Inline JavaScript pomocí např. tagu onclick
<button onclick="alert('Hello!')">Klikni mě</button>
// 3. Externí JavaScript soubor
<script src="script.js"></script>
// 4. Přidání JavaScriptu dynamicky pomocí DOM
let script = document.createElement('script'); script.src = 'script.js'; document.body.appendChild(script);
// 5. Použití JavaScriptu přes URL v href (Javascript URI)
<a href="javascript:alert('Hello!')">Klikni zde</a>

4: Jak je možné v JavaScriptu deklarovat proměnné?

Uveďte na příkladech různé datové typy, které je možné využívat v JS.

Doplňte chybějící kód/text podle vzoru úvodních dvou řádků!

let x = 0; // s přiřazením hodnoty
let x; // bez přiřazení hodnoty
let x = null; // s hodnotou null (prázdná hodnota)
let x = 5; // celé číslo (typ number)
let x = 3.14; // reálné číslo (typ number)
let x = 1.5e6; // zápis desetinného čísla pomocí exponentu
let text = "Ahoj svět"; // typ string
let prestupny = false; // logická hodnota (typ boolean)

5: Jaké operátory můžeme používat v JavaScriptu?

Vypište operátory, které patří do uvedených skupin.

U méně obvyklých operátorů dopište do závorky jejich význam!

Matematické

+, -, *, /, % (modulo - zbytek po celočíselném dělení), ** (umocnění)

Přiřazovací

+=, -=, *=, /=, %=, **=

Logické

&& : AND (a zároveň), || : OR (nebo), ! : NOT (negace)

Porovnávací

== : Rovnost, === : Přísná rovnost, != : Nerovnost, !== : Přísná nerovnost, > : Větší než, < : Menší než, >= : Větší nebo rovno, <= : Menší nebo rovno

Bitové

& : Bitový AND, | : Bitový OR, ^ : Bitový XOR, ~ : Bitový NOT, << : Posun bitů doleva, >> : Posun bitů doprava (s ohledem na znaménko), >>> : Posun bitů doprava bez ohledu na znaménko

Typové

typeof (typ proměnné), instanceof ( Zjišťuje, zda objekt je instancí určité třídy)

6: Která popup okna lze využít v JavaScriptu?

K uvedeným příkladům připište do poznámky, co se stane na webové stránce.

Na konci doplňte aspoň dvě varování, proč bychom se měli vyhýbat použití popup oken v reálných webových aplikacích.

alert()

/* Co se stane: Zobrazí jednoduché popup okno s textovou zprávou a tlačítkem OK. Uživatel musí potvrdit tuto zprávu kliknutím na tlačítko OK, aby mohl pokračovat v interakci se stránkou. */

alert('Nezadal jsi správné heslo!');

prompt()

/* Do proměnné password vloží heslo zadané uživatelem */

Co se stane: Otevře popup okno s textovým polem, kam může uživatel zadat text. Zadaná hodnota se uloží do proměnné. Popup okno také obsahuje tlačítka OK a Zrušit. Pokud uživatel klikne na Zrušit, funkce vrátí hodnotu null.

confirm()

/* Co se stane: Zobrazí popup okno s textovou zprávou a dvěma tlačítky: OK a Zrušit. Funkce vrací true, pokud uživatel klikne na OK, a false, pokud klikne na Zrušit. V tomto případě, pokud uživatel klikne na OK, bude přesměrován na stránky Seznam.cz*/

if (confirm('Chceš přejít na stránky Seznam.cz?')) location.href='https://seznam.cz';
